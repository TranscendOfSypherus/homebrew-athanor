# homebrew-athanor

Homebrew tap for [Athanor](https://github.com/TranscendOfSypherus/athanor-releases) — a
web-based platform for managing agentic coding sessions.

```sh
brew install transcendofsypherus/athanor/athanor
```

Works on macOS (Apple Silicon), Linux, and WSL2. The formula installs the prebuilt binary
from the [public releases](https://github.com/TranscendOfSypherus/athanor-releases/releases)
and is regenerated automatically by the release workflow — manual edits will be overwritten.
